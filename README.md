🎓 Elite-LMS: A Premium Learning Experience
Elite-LMS is a high-performance, community-driven learning management system. It’s designed not just to host videos, but to keep students engaged through gamification, real-time interaction, and a sleek "Elite" UI.

🔥 Key Features
🌓 Adaptive Dark Mode
A seamless theme engine that respects user preference. Includes a dynamic toggle that swaps icons and text labels ("Dark Mode" vs "Light Mode") and saves your choice across sessions.

🏆 Gamification & Streaks
Daily Streaks: Tracks user consistency. Don't break the chain, vro!

Dynamic Badges: Automatic role labeling in comments:

VERIFIED TEACHER: Blue gradient for instructors.

GRADUATE: For students who conquered the final quiz.

PRO STUDENT: For those crushing the current curriculum.

LEARNING: Real-time status for active watchers.

💬 Community Discussion
A Reddit-style threaded comment system under every lesson. Supports nested replies, instructor-verified highlights, and reporting for a clean learning environment.

📝 Integrated Student Workspace
Premium Notepad: A dedicated tab for taking notes while watching videos.

Downloadable Content: Export your notes as .txt files with one click.

Watermarked Player: Anti-piracy watermarks displaying the student's username over the video.

🛠 Tech Stack
Python / Django: Robust backend logic.

Bootstrap 5.3: Modern, responsive UI components.

JavaScript (Vanilla): Dynamic theme switching and UI interactions.

HTML5/CSS3: Custom "Elite" styling and glassmorphism.
